# Superstore Sales Analysis - Power BI

## Project Overview
This project analyzes the sales performance of a Global Superstore to identify key trends, top-selling products, and regional profitability.

## Tools Used
* **Power BI:** Data Cleaning, Modeling, and Visualization.
* **DAX:** Created custom measures for Profit Margin and YoY Growth.
* **Excel:** Data Source.

## Key Insights
* **Top Category:** Technology had the highest sales growth.
* **Profitability:** Western region outperformed other regions in profit margins.
* **Shipping:** Standard Class is the most preferred shipping mode.
